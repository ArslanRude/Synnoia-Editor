# Synnoia 📝🤖
**AI-Powered Document Editor with Agentic Intelligence**

---

## 📌 Overview
Synnoia is an advanced **AI-powered document editor** designed to provide a seamless writing experience.  
It combines the familiarity of a **Word-like editor** with the intelligence of **AI agents**, enabling smart assistance, line completions, document analysis, and context-aware writing suggestions.  

The name **Synnoia** comes from the Greek word "σύνοια" meaning *shared understanding or mutual thought*, reflecting its purpose: **to co-create documents with human-like intelligence**.

---

## ✨ Key Features
- **Word Processing Clone**  
  Full-featured document editor built with **Vue.js & TypeScript**, replicating MS Word’s core functionalities.

- **Smart AI Agent System**  
  AI agent capable of **reading, writing, and analyzing** documents in real time.

- **Line Completion**  
  Intelligent AI-assisted full line suggestions for faster writing.

- **Context-Aware Prompts**  
  Smart prompting system for summaries, corrections, rewriting, and generating content.

- **Database & Security**  
  Secure storage system and user authentication for safe document handling.

---

## 🛠️ Tech Stack
- **Frontend:** Vue.js, TypeScript  
- **Backend:** Node.js / Express.js  
- **Database:** MongoDB / SQL (final decision based on testing)  
- **AI Models:** LLMs integrated via LangChain / OpenAI APIs  
- **Security:** JWT Authentication, Database Security Layer
